# 📊 CareerBridge Admin Analytics Dashboard

A modern responsive Admin Analytics Dashboard built using React.js for the CareerBridge Job Portal platform.
This dashboard enables employers/admins to manage jobs, monitor applications, track users, and visualize analytics dynamically with charts and localStorage-based CRUD operations.

📌 Note: This Admin Analytics Dashboard is an extended module of the CareerBridge Job Portal developed in Week 2 Task 1 and enhanced further with analytics, dashboard management, charts, and admin functionalities.
This project was developed as part of my Web Development Internship at Sqrock IT Solutions. 🚀

> ⚠️ Note: All company names, user information, jobs, applications, analytics data, and dashboard content used in this project are completely dummy/demo data created only for learning and educational purposes. Please do not share or misuse any information shown in this project.

4. Click Deploy

# ⚙️ Setup Instructions

### Clone Repository
```bash
git clone https://github.com/your-github-username/careerbridge-admin-dashboard.git
```

### Install Dependencies
```bash
npm install
```

### Run Project
```bash
npm run dev
```

Project will run at:
```bash
http://localhost:5173/
```
